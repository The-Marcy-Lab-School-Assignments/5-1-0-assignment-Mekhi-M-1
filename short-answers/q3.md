# Question

In a code snippet, demonstrate a `User` class with at least one:

- class property
- class method
- instance property
- instance method

# Answer

```js
class User {
  static #all = [];

  constructor(name) {
    this.name = name;

    User.#all.push(this);
  }

  greet() {
    console.log(`Hello! My name is ${this.name}`);
  }

  static getAllUsers() {
    return [...User.#all];
  }
}
```
